# Econometria com Dados do Gujarati - UFPR

Análise de regressão aplicada aos dados do livro Basic Economdtrics (Gujarati), desenvolvida na disciplina de Econometria 2 da UFPR.

## Dados
Dados de exemplo do Gujarati, séries econômicas usadas para estudo econometrico.

## O que o modelo mostra
Estimação de coeficientes, interpretação de resultados e diagnóstico dos resíduos.
